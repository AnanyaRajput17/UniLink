# UniLink

##  Project Overview
UniLink is a collaborative platform designed to connect users and manage teams efficiently.

---

##  System Architecture

The project follows a 3-layer architecture:

### 1. Frontend (User Interface)
- Built using HTML5, CSS, or Vaadin
- Responsible for user interaction

### 2. Backend (Application Layer)
- Built using Spring Boot (Java)
- Handles core logic:
  - Authentication Service
  - Team Management (Create/Join)
  - Profile Management

### 3. Database Layer
- Uses Relational Database:
  - MySQL / PostgreSQL
- Stores user and team data

### 4. External APIs
- Integration with:
  - GitHub
  - LinkedIn

---

##  Workflow
User → Frontend → Backend → Database  
Backend ↔ External APIs
